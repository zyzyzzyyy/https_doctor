# HTTPS 证书检测工具 - 规格说明书

## 1. 项目概述

### 项目名称
HTTPS Doctor（https_doctor）

### 项目类型
桌面工具 - HTTPS 证书状态检测与诊断

### 核心功能
输入 HTTPS URL，自动检测证书链状态、过期时间、吊销状态、域名匹配、TLS 版本等信息，并生成 HTML 报告。

### 目标用户
- 开发者本人（服务器日常巡检）
- 不熟悉 HTTPS 证书的团队成员（问题诊断）

---

## 2. 用户故事

### 场景一：单站点检测
作为运维人员，我发现某个 HTTPS 站点无法访问，想快速诊断是否是证书问题。
1. 打开工具，输入 URL `https://example.com`
2. 点击"开始检测"
3. 立即看到结果：证书是否有效、过期时间、证书链是否完整

### 场景二：批量检测
作为安全负责人，我要检查所有子域名的证书状态。
1. 从监控平台导出 URL 列表
2. 粘贴到工具的多行文本框
3. 批量检测，一目了然看到哪些站点有问题

### 场景三：导出报告
作为技术支持，我要给客户发送诊断报告。
1. 完成检测后
2. 点击"导出 HTML 报告"
3. 生成包含详细信息的 HTML 文件，可以直接发送

## 3. 功能需求

### 3.1 证书状态检测

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 过期检测 | 检查证书是否过期，显示距离过期的天数 | 必须 |
| 证书链完整性 | 检查 Root CA → Intermediate CA → Server Cert 是否完整 | 必须 |
| 证书链详情 | 显示证书链中每个证书的名称（CN/ON） | 必须 |
| 吊销状态 | 实时联网查询 OCSP，判断证书是否被吊销 | 必须 |
| 域名匹配 | 验证证书的 CN/SAN 是否与访问域名匹配 | 必须 |
| TLS 版本 | 显示服务器当前使用的 TLS 版本 | 必须 |
| 加密算法 | 显示支持的密钥交换和通讯加密算法 | 必须 |

### 3.2 输入方式

| 方式 | 说明 |
|------|------|
| 单个 URL | 直接在文本框输入一个 URL |
| 批量 URL | 多行粘贴，每行一个 URL |

### 3.3 输出方式

| 方式 | 说明 |
|------|------|
| 界面表格 | 实时显示所有 URL 的检测结果摘要 |
| 详情面板 | 选中某条结果后，显示完整的证书链、OCSP、域名验证等详细信息 |
| HTML 报告 | 导出包含完整检测结果的 HTML 文件 |

### 5.2 状态标识

| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 有效 | ✓ | 绿色 | 所有检测项通过 |
| 警告 | ⚠ | 黄色 | 有非关键问题（如证书快过期、TLS 版本较旧） |
| 失败 | ✗ | 红色 | 关键问题（证书过期、域名不匹配、证书链不完整） |
| 检测中 | ⟳ | 蓝色 | 正在检测 |
| 连接失败 | ✗ | 灰色 | 无法连接到服务器 |

### 5.3 证书链列显示格式

| 情况 | 显示内容 |
|------|----------|
| 完整 | `完整` |
| 不完整 | `不完整 (缺少: Intermediate CA)` |

## 6. 技术架构

### 6.1 模块划分

```

├── cert_checker.py       # 核心检测逻辑（纯函数，无 GUI 依赖）

├── gui.py                # wxPython GUI 主程序

```

### 6.2 cert_checker.py 接口设计

```python
def check_certificate(url: str) -> dict:
    """
    检测单个 URL 的证书状态
    返回结构:
    {
        "url": str,

        "cert_chain": [

```

## 7. 错误处理

| 场景 | 处理方式 | 用户看到的提示 |
|------|----------|---------------|
| 无法解析域名 | 标记连接失败 | `✗ 连接失败: 无法解析域名` |
| 连接超时 | 标记连接失败 | `✗ 连接失败: 连接超时` |
| 连接被拒绝 | 标记连接失败 | `✗ 连接失败: 连接被拒绝` |
| 证书链不完整 | 标记警告 | `⚠ 证书链不完整 (缺少: ...)` |
| 证书过期 | 标记失败 | `✗ 证书已过期` |
| 域名不匹配 | 标记失败 | `✗ 域名不匹配: cert=xxx,访问=yyy` |
| OCSP 查询失败 | 标记警告 | `⚠ 无法验证吊销状态` |
| TLS 版本过旧 | 标记警告 | `⚠ TLS 1.0 已弃用` |
